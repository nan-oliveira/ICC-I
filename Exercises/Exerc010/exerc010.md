Escreva um programa que recebe 3 valores e escreva os valores em ordem crescente.

Entrada: Três números inteiros

Saida: Os três númeors inteiros ordenados de forma crescente.

## Exemplos de Entrada e Saída

Entrada:

```
3 2 1
```

Saída:

```
1 2 3
```

Entrada:

```
4 5 2
```
Saída:
```
2 4 5
```
